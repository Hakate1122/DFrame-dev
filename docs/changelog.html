Changelog

Mọi thay đổi đáng chú ý của DFrame sẽ được ghi lại ở đây.

2026.4.8-dev 2026-04-08

Added

  • dli version có thể (tùy chọn) hiển thị GitHub/Git source info (origin remote URL, current branch, short commit hash) khi chạy trong thư mục có .git/. Lệnh sẽ hỏi người dùng trước khi in để tránh làm ồn output mặc định.
  • dli tikitiki: bot/wizard hỗ trợ khởi tạo project DFrame mới (copy starter template, tạo .env, tùy chọn copy vendor/).

Changed

  • Chạy DFrame từ PHAR không còn bắt buộc phải có file .env cạnh/ở trong PHAR. Khi chạy từ PHAR, cơ chế load env sẽ kiểm tra thêm current working directory (thư mục bạn chạy lệnh), và tiếp tục chạy kể cả khi không tìm thấy .env (để CLI/report/help vẫn hoạt động).
